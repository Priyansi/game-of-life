# Game Of Life

Conway's Simulation called Game of Life implemented using Pygame.

1. Clone the repo.
2. Launch the executable.
3. Navigation -
  - `space` for start/stop.
  - &#8593; for zoom in.
  - &#8595; for zoom out.
  - &#8592; for shifting view to left.
  - &#8594; for shifting view to right.
4. Watch the magic.
